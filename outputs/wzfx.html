元分析主模型结果解读

本研究采用分层混合效应模型，以 Fisher’s z 为效应量，检验空间尺度对本地–外来物种丰富度关系（NERR）的影响。 结果显示：grain 的对数与效应量呈显著正相关，支持原文核心结论——取样面积越大，本地与外来物种丰富度正相关越强。 整体模型截距为正，说明在所有尺度上平均均为正相关，不支持传统“入侵悖论”。

random <- list(~ 1 | Study / Case)
metar <- rma.mv(yi = z, V = var_z, mods = ~ ln_Grain, random = random, data = dat, method = "ML")

异质性分析解读

小 grain 下效应量异质性极高，正负相关并存；随 grain 增大异质性降低，效应趋于稳定正值。 说明小尺度受局部生物互作与随机波动影响大，而大尺度更能体现环境驱动的全局格局。

Fig.S1作图

漏斗图绘制说明

本代码基于metafor包实现元分析漏斗图绘制，用于识别研究间的发表偏倚。 1. 分别拟合多水平元分析模型（randomonly）与常规随机效应模型（randomonly2）； 2. 绘制带90%/95%/99%置信区间的轮廓增强漏斗图，以标准误为纵轴； 3. 添加红色实线标记合并效应量位置，直观展示效应量分布与偏倚特征。

randomonly <- rma.mv(yi = z, V = var_z, random = random, data = dat, method = "REML")
randomonly2 <- rma(yi = z, vi = var_z, data = dat)
funnel(randomonly2, level=c(90, 95, 99),
       shade=c("white", "gray", "darkgray"), refline=0,
       main="Standard Error")
abline(v=randomonly$b, lty="solid",col="red",lwd=2)

漏斗图展示了效应量围绕均值的分布情况。其中，垂直线代表基于随机效应模型估计的合并效应量；漏斗区域以 0 为中心。统计显著性区域由阴影区域表示：白色区域代表 p > 0.10，深灰色区域代表 0.05 < p ≤ 0.10，中灰色区域代表 0.01 < p ≤ 0.05，漏斗外的浅灰色区域代表 p ≤ 0.01。

图表解读（Figure 4a）

Figure 4a 展示了本元分析所纳入研究的全球空间分布格局。 图中灰色底纹为全球大陆轮廓，紫色点代表每一个研究的野外调查/实验地点。

从空间分布可以看出： 1. 研究分布极不均匀，绝大多数研究集中在北半球温带区域，包括欧洲、北美洲。 2. 南半球研究数量明显偏少，主要分布在南美洲南部与澳大利亚东部。 3. 非洲、亚洲大部分地区、极地地区研究极度匮乏，存在明显的地理研究偏倚。 4. 这种分布特征表明，当前关于本地物种与入侵物种丰富度关系的研究结论，更适用于北半球温带地区，对热带、亚热带及极地地区的外推需要谨慎。

图注

Figure 4a. Global distribution of sampling sites for studies included in the meta-analysis. Each point represents the geographic location of one study. The distribution shows a strong geographic bias, with most studies concentrated in temperate zones of the Northern Hemisphere.
